# 2048 数字游戏

一个基于HTML5、CSS3和JavaScript实现的2048益智游戏。

## 游戏特色

- 完整的2048游戏逻辑
- 响应式设计，支持电脑和手机
- 三种游戏模式：4×4、5×5、6×6
- 游戏统计（移动次数、游戏时间、最大方块）
- 撤销功能
- 触摸控制（针对移动设备）
- 本地存储最高分
- 精美的UI设计和动画效果

## 如何运行

1. 直接将文件上传到GitHub仓库
2. 启用GitHub Pages
3. 访问生成的链接即可开始游戏

## 游戏控制

- **电脑**: 使用键盘方向键(↑, ↓, ←, →)移动方块
- **手机/平板**: 使用屏幕下方的触摸按钮
- **快捷键**:
  - `Ctrl + R`: 重新开始游戏
  - `Ctrl + Z`: 撤销上一步

## 部署到GitHub Pages

1. 在GitHub上创建新仓库
2. 上传所有文件
3. 进入仓库设置 → Pages
4. 选择main分支作为源
5. 保存设置，等待部署完成

## 文件说明

- `index.html`: 主HTML文件
- `style.css`: 样式文件
- `script.js`: JavaScript游戏逻辑
- `README.md`: 项目说明文档

## 游戏规则

1. 使用方向键移动所有方块
2. 当两个相同数字的方块碰撞时，它们会合并成它们的和
3. 每次移动后，会在空白处生成一个新的方块（2或4）
4. 游戏目标：合并出2048方块
5. 游戏结束条件：网格被填满且无法进行任何合并

## 自定义

你可以通过修改以下内容来自定义游戏：
- `style.css` 中的颜色变量
- `script.js` 中的游戏配置
- 添加新的游戏模式
